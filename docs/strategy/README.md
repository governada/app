## How to Use

- `ultimate-vision.md` is the **north star** - read it first for any strategic decision.
- `monetization-strategy.md` has detailed business model, pricing tiers, and revenue projections that complement the vision doc's summary.
- `catalyst-proposal.md` is the active Catalyst Fund 16 submission draft.
- `plans/governada-platform-2026.md` is the active platform initiative for stack, infra, design-system split, and Studio architecture decisions.
- For active platform execution, follow `plans/governada-platform-2026.md` with `specs/governada-platform-phase0-workplan.md`, `specs/governada-platform-runtime-safety.md`, and `specs/governada-platform-architecture-inventory.md`.
- Agent cheat sheet: `.cursor/rules/strategy.md` has a condensed 80-line extract of the vision for quick agent context loading.
