# DRepScore Strategy Documents

Foundational strategy documents for DRepScore monetization, growth, and ecosystem expansion.

## North Star

**[ultimate-vision.md](ultimate-vision.md)** is the definitive product vision. It defines the build sequence, monetization phases, data compounding strategy, and the interconnected system design. All build decisions, feature prioritization, and architecture choices should align with this document.

## How to Use

- `ultimate-vision.md` is the **north star** -- read it first to understand where the product is going and why.
- `monetization-strategy.md` has the detailed business model, pricing tiers, and revenue projections.
- Sub-documents go deep on specific revenue streams (DRep Pro, API, Catalyst).
- The `product-wow-plan-*.md` files are historical -- they document what was built in Sessions 12-19 but are superseded by the ultimate vision for future direction.
- Each sub-doc tracks its own status, open questions, and next actions.
