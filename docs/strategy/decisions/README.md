# Decision Log

Strategic decisions captured from `/strategy` sessions. Each file is a single decision with context, options considered, and rationale.

## Naming Convention

`YYYY-MM-DD-[slug].md` — e.g., `2026-03-15-monetization-timing.md`

## Template

```markdown
# Decision: [Title]

Date: YYYY-MM-DD
Status: DECIDED | REVISIT BY [date] | SUPERSEDED BY [ref]
Context: [1-2 sentences — what prompted this decision]

## Question

[The specific question being decided]

## Options Considered

### Option A: [Name]

- [Pro 1]
- [Pro 2]
- [Con 1]
- [Con 2]

### Option B: [Name]

- [Pro 1]
- [Pro 2]
- [Con 1]
- [Con 2]

## Decision

[Which option, stated clearly]

## Rationale

[Why this option. Reference principles, data, competitive positioning, user impact.]

## Consequences

- [What this means for the roadmap]
- [What we're giving up]
- [When to revisit this decision]

## Related

- [Links to relevant vision sections, PRs, issues, other decisions]
```
